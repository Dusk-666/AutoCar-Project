# AutoCar-Project 🚗

基于STM32F103C8T6的智能循迹小车项目，采用PD控制算法和状态机实现平滑稳定的循迹功能。

## 📋 项目特性

- ✅ **双环PD控制** - 外环位置控制（红外传感器），内环速度控制（霍尔编码器）
- ✅ **状态机设计** - 智能处理直行、转弯、十字路口等场景
- ✅ **速度比例控制** - 自动调整左右轮速度比例（1.778:1）//经过多组平行测试得出，比例为16:9
- ✅ **速度渐变** - 平滑的速度变化，避免突变
- ✅ **十字路口处理** - 自动减速通过十字路口
- ✅ **直角转弯** - 检测并处理90度直角弯道
- ✅ **脱线恢复** - 脱线后保持前进，自动恢复循迹
- ✅ **详细注释** - 中文注释，易于理解和修改

## 🎮 功能说明

### 1. 双环PD控制

#### 外环位置控制 (TRACKING状态)

使用PD算法实现平滑循迹：

```
转向量 = Kp × 误差 + Kd × 误差变化率
左轮目标速度 = 基础速度 - 转向量
右轮目标速度 = 基础速度 + 转向量
```

**误差定义**：
- 采用加权平均算法，根据传感器位置计算误差
- 误差范围：-3.0（最左）到 +3.0（最右）
- 0.0表示完美居中

#### 内环速度控制

使用霍尔编码器反馈实现速度闭环控制：

```
速度误差 = 目标速度 - 实际速度
PWM输出 = 目标速度 + Kp × 速度误差 + Kd × 速度误差变化率
```

**速度计算**：
- 基于编码器计数变化
- 单位：脉冲/控制周期
- 控制周期：5ms

### 2. 大角度转弯 (TURN\_LEFT/TURN\_RIGHT状态)

当外侧传感器检测到黑线时，进入转弯状态：

**左转**：

- 左轮速度 = 基础速度 × 40%
- 右轮速度 = 基础速度 × 130%

**右转**：

- 左轮速度 = 基础速度 × 130%
- 右轮速度 = 基础速度 × 40%

### 3. 十字路口处理

检测到全黑时：

- 降低速度到设定比例
- 保持直行通过
- 等待传感器恢复正常

### 4. 脱线处理

检测到全白时：

- 保持当前速度继续前进
- 等待找到黑线后恢复循迹

### 5. 速度保护

- **最小速度**：50（确保电机始终转动）
- **最大速度**：999（PWM上限）
- **速度渐变**：每次调整步长为20，实现平滑过渡

## 📊 调试建议

### 1. 外环位置控制参数调试

1. **初始参数**：Kp=90, Kd=38
2. **震荡问题**：减小Kp，增大Kd
3. **响应慢**：增大Kp
4. **过冲问题**：增大Kd

### 2. 内环速度控制参数调试

1. **初始参数**：Kp=0.5, Kd=0.1
2. **速度波动大**：减小Kp，增大Kd
3. **速度响应慢**：增大Kp
4. **电机噪音大**：减小Kp和Kd

### 3. 转弯参数调试

1. **转弯不够**：减小TURN_INNER_RATIO，增大TURN_OUTER_RATIO
2. **转弯太急**：增大TURN_INNER_RATIO，减小TURN_OUTER_RATIO
3. **误触发**：增大OUTER_TRIGGER_TURN_COUNT

### 4. 系统测试

1. **直道测试**：观察是否稳定直行，速度是否稳定
2. **小弯测试**：观察是否能平滑修正，速度是否保持稳定
3. **大弯测试**：观察外侧传感器触发和转弯效果
4. **十字路口测试**：观察是否能顺利通过
5. **速度响应测试**：观察速度变化是否平滑，是否有震荡

## 🔧 核心算法

### 状态机流程

```
TRACKING (PD控制)
    │
    ├─ 左外传感器触发 ─→ APPROACH ─→ TURN_LEFT (左转)
    │                                    │
    │                                    └─ 恢复居中 → TRACKING
    │
    ├─ 右外传感器触发 ─→ APPROACH ─→ TURN_RIGHT (右转)
    │                                    │
    │                                    └─ 恢复居中 → TRACKING
    │
    ├─ 全黑检测 → 减速直行通过
    │
    └─ 全白检测 → 保持速度前进
```

### PD控制公式

```
误差计算：
  左内黑 + 右内黑 → error = 0
  左内黑 + 右内白 → error = -2
  左内白 + 右内黑 → error = +2

转向量计算：
  steer = Kp × error + Kd × (error - last_error)
  steer = clamp(steer, -MAX_STEER, +MAX_STEER)

速度计算：
  左轮速度 = 基础速度 + steer
  右轮速度 = 基础速度 - steer
```
