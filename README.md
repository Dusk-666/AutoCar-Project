# AutoCar-Project 🚗

基于STM32F103C8T6的智能循迹小车项目，采用PD控制算法和状态机实现平滑稳定的循迹功能。

## 📋 项目特性

- ✅ **PD控制算法** - 平滑的循迹控制，避免震荡
- ✅ **状态机设计** - 智能处理直行、转弯、十字路口等场景
- ✅ **速度比例控制** - 自动调整左右轮速度比例（1.778:1）//经过多组测试得出，比例为16:9
- ✅ **速度渐变** - 平滑的速度变化，避免突变
- ✅ **十字路口处理** - 自动减速通过十字路口
- ✅ **直角转弯** - 检测并处理90度直角弯道
- ✅ **脱线恢复** - 脱线后保持前进，自动恢复循迹
- ✅ **详细注释** - 中文注释，易于理解和修改

### 传感器逻辑

- **检测到黑线**：传感器输出 `0` (SENSOR\_BLACK)
- **检测到白底**：传感器输出 `1` (SENSOR\_WHITE)

## 🎮 功能说明

### 1. PD控制循迹 (TRACKING状态)

使用PD算法实现平滑循迹：

```
转向量 = Kp × 误差 + Kd × 误差变化率
左轮速度 = 基础速度 - 转向量
右轮速度 = 基础速度 + 转向量
```

**误差定义**：

- `左内黑 + 右内黑` → 误差 = 0（完美居中）
- `左内黑 + 右内白` → 误差 = -2（偏右，需要左转）
- `左内白 + 右内黑` → 误差 = +2（偏左，需要右转）

### 2. 大角度转弯 (TURN\_LEFT/TURN\_RIGHT状态)

当外侧传感器检测到黑线时，进入转弯状态：

**左转**：

- 左轮速度 = 基础速度 × 40%
- 右轮速度 = 基础速度 × 130%

**右转**：

- 左轮速度 = 基础速度 × 130%
- 右轮速度 = 基础速度 × 40%

### 3. 十字路口处理

检测到全黑时：

- 降低速度到设定比例
- 保持直行通过
- 等待传感器恢复正常

### 4. 脱线处理

检测到全白时：

- 保持当前速度继续前进
- 等待找到黑线后恢复循迹

### 5. 速度保护

- **最小速度**：50（确保电机始终转动）
- **最大速度**：999（PWM上限）
- **速度渐变**：每次调整步长为20，实现平滑过渡

## 📊 调试建议

### 1. PD参数调试

1. **初始参数**：Kp=40, Kd=25
2. **震荡问题**：减小Kp，增大Kd
3. **响应慢**：增大Kp
4. **过冲问题**：增大Kd

### 2. 转弯参数调试

1. **转弯不够**：减小TURN\_INNER\_RATIO，增大TURN\_OUTER\_RATIO
2. **转弯太急**：增大TURN\_INNER\_RATIO，减小TURN\_OUTER\_RATIO
3. **误触发**：增大OUTER\_SENSOR\_CONFIRM

### 3. 速度调试

1. **直道测试**：观察是否稳定直行
2. **小弯测试**：观察是否能平滑修正
3. **大弯测试**：观察外侧传感器触发和转弯效果
4. **十字路口测试**：观察是否能顺利通过

## 🔧 核心算法

### 状态机流程

```
TRACKING (PD控制)
    │
    ├─ 左外传感器触发 ─→ APPROACH ─→ TURN_LEFT (左转)
    │                                    │
    │                                    └─ 恢复居中 → TRACKING
    │
    ├─ 右外传感器触发 ─→ APPROACH ─→ TURN_RIGHT (右转)
    │                                    │
    │                                    └─ 恢复居中 → TRACKING
    │
    ├─ 全黑检测 → 减速直行通过
    │
    └─ 全白检测 → 保持速度前进
```

### PD控制公式

```
误差计算：
  左内黑 + 右内黑 → error = 0
  左内黑 + 右内白 → error = -2
  左内白 + 右内黑 → error = +2

转向量计算：
  steer = Kp × error + Kd × (error - last_error)
  steer = clamp(steer, -MAX_STEER, +MAX_STEER)

速度计算：
  左轮速度 = 基础速度 + steer
  右轮速度 = 基础速度 - steer
```
